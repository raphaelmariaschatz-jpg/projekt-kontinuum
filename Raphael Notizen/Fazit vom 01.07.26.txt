Eine kleine Ergänzung zu Git

Im Bericht steht mehrfach:

git status meldet weiterhin, dass C:\Projekt Kontinuum kein Git-Repository ist.

Das ist kein Fehler des GitAgenten – im Gegenteil: Er erkennt den Zustand korrekt. Bevor ihr den CGM im Alltag nutzt, solltet ihr das Projektverzeichnis einmal mit Git initialisieren (git init) oder ein bestehendes Repository verwenden. Erst dann kann der GitAgent seine eigentliche Stärke entfalten.
