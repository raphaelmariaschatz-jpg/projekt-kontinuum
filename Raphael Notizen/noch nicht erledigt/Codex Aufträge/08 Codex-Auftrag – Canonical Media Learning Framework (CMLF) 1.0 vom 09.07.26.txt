# Codex-Auftrag – Canonical Media Learning Framework (CMLF) 1.0

## Ziel des Auftrags

Bitte prüfe und konzipiere das neue Architekturkonzept:

```text
Canonical Media Learning Framework (CMLF) 1.0
```

Das Canonical Media Learning Framework soll den kanonischen Rahmen definieren, über welche Medien, Darstellungsformen und Lernmethoden Projekt Kontinuum Wissen und Kompetenzen vermittelt.

Das CMLF beschreibt nicht die Inhalte des Lernens, sondern die geeigneten Lernmedien und Vermittlungsformen.

---

# Grundsatz

Projekt Kontinuum soll Wissen so vermitteln, dass Menschen entsprechend ihrer individuellen Lernweise möglichst effektiv lernen können.

Dabei gilt:

* Nicht jedes Thema eignet sich für dieselbe Darstellungsform.
* Nicht jeder Mensch lernt auf dieselbe Weise.
* Die Wahl des Mediums ist Teil des Lernprozesses.

---

# Zielsetzung

Das CMLF soll festlegen,

* welche Medientypen unterstützt werden,
* wann welche Darstellungsform sinnvoll ist,
* wie unterschiedliche Medien kombiniert werden,
* wie adaptive Medienempfehlungen entstehen,
* und wie Lernfortschritte medienübergreifend unterstützt werden.

* wie individuelle Präferenzen berücksichtigt werden können.

In dieser Phase erfolgt **keine Implementierung**.

Bitte:

1. analysiere das Konzept
2. prüfe die Einordnung in die Canonical Architecture
3. definiere die Medienbereiche

5. identifiziere notwendige Schnittstellen

---

# Leitplanken

Erlaubt:

* Architekturkonzept
* Dokumentation
* JSON-Schema
* Statusbericht
* Glossar-Ergänzung
* Implementierungsplan

Nicht erlaubt:

* Runtime-Änderungen
* Änderungen an CRE, Execution Planner oder Orchestrator Core
* neue Agentenimplementierungen
* Datenbankmigrationen
* automatische Mediengenerierung im Produktivbetrieb
* Refactorings bestehender Komponenten

---

# Entscheidungskriterium

Das Canonical Media Learning Framework soll nur dann als GO bewertet werden, wenn es:

* die Lernarchitektur sinnvoll ergänzt,
* CAICF unterstützt, aber nicht ersetzt,
* die CCP bei der Ausgabe geeigneter Lernmedien unterstützt,
* unterschiedliche Lernstile berücksichtigt,
* langfristig erweiterbar bleibt,
* die Architektur nicht unnötig verkompliziert.
