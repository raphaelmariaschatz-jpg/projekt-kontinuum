Warum ich "Vision" vom Intelligence Framework trennen würde

Das Canonical Intelligence Framework (CIF) beantwortet Fragen wie:

Welche Modelle stehen zur Verfügung?
Wann wird welches Modell verwendet?
Wie werden Modelle bewertet?
Wie werden Ergebnisse validiert?

Das Canonical Vision Framework (CVF) beantwortet dagegen ganz andere Fragen:

Wie gelangen visuelle Informationen in K?
Wie werden Bilder beschrieben?
Wie werden Objekte erkannt?
Wie werden mehrere Bilder miteinander verglichen?
Wie wird räumlicher Kontext aufgebaut?
Wie werden visuelle Erkenntnisse im Memory gespeichert?

Das sind zwei verschiedene Verantwortlichkeiten.
