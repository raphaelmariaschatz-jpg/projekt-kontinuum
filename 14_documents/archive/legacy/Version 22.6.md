# Projekt Kontinuum 22.6 – Vollversion

Dies ist die offizielle Vollversion 22.6 von Projekt Kontinuum.

Diese Version ersetzt die vorherigen Zwischenstände 22.6.1 und 22.6.2 als offizielle Versionslinie.
Die Entwicklung geht danach weiter mit 22.7, 22.8 usw.

## Hauptänderungen in Version 22.6

* Vollversion, kein Patch
* GUI mit Login-Maske
* Benutzername/Passwort-System mit Hash-Prüfung
* Wiederherstellungsschlüssel-System vorbereitet
* Natürliches Fragen ohne Befehlszwang
* Cognitive Core mit Agentenstruktur
* Suchmonitor-Fenster: Raphael sieht live, wo Kontinuum sucht
* Keine Anzeige technischer Ordnerstrukturen im Dialog
* Identitätskern:

  * Raphael Schatz ist der Erschaffer von Projekt Kontinuum
  * Grundprinzip: Erkennen – Schaffen – Vollenden
  * Grundprinzip: Der Weg ist das Ziel
* Memory Agent
* Knowledge Agent
* Research Agent
* Learning Agent
* Calculator Agent
* Project Chronicle Manager
* Quellenbewertung vorbereitet
* Internet-, Universitäts- und Bibliothekszugang als erweiterbare Schnittstellen vorbereitet

## Start

```bash
python main.py
```

## Testzugang

Benutzername:

```text
Raphael
```

Passwort:

```text
kontinuum
```

Wichtig: In einer echten lokalen Installation sollte das Passwort neu gesetzt werden.



## Windows-Startdateien

Diese Version enthält zusätzlich:

* `START\\\_KONTINUUM\\\_22\\\_6.bat`
* `INSTALL\\\_REQUIREMENTS\\\_22\\\_6.bat`
* `RESET\\\_LOGIN\\\_22\\\_6.bat`
* `TEST\\\_KONTINUUM\\\_22\\\_6.bat`

Für normalen Start unter Windows genügt ein Doppelklick auf:

```text
START\\\_KONTINUUM\\\_22\\\_6.bat
```
