# Kontinuum 30.0 - Meaning Core / Bedeutungsgraph

Datum: 2026-06-16

Kontinuum 30.0 führt den Meaning Core ein. Die neue Schicht verbindet
Prinzipien, Ziele, Handlungen, Erinnerungen, Chronik und Identität zu einem
Bedeutungsgraphen.

## Eingeführt

- Meaning Core
- Tabellen `meaning_nodes`, `meaning_edges`, `meaning_paths`
- kanonischer Pfad: Prinzip -> Ziel -> Handlung -> Erinnerung -> Chronik -> Identität
- Befehle `bedeutungsstatus` und `bedeutungspfad`
- Selbstmodellbeobachtung der Bedeutungsschicht
- append-only-Schutz für Bedeutungsbeziehungen und Bedeutungspfade
- GUI-Schnellbefehle für Meaning Core
